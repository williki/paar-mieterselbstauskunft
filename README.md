# Paar Mieterselbstauskunft
Deutsche LaTeX Templates für eine Mieterselbstauskunft für Paare.

## Anpassen
Persönliche Daten in vars.tex eintragen. Bilder in img/ ablegen.

## Build
`make $TYP`
oder einfach `make` um alle zu bauen.
